normal command
-------------------------------------------------------------------------------------
J	将两行合并成一行
u	撤消
CTRL-R	重做
U	撤消对此行的修改
i	插入
a	在行尾追加
o	在当前行下面另起一行并使用和当前行相同的缩进
O	在当前行上面另起一行并使用和当前行相同的缩进
ZZ	保存并退出
CTRL-]	跳转到标签
CTRL-T	返回上一个标签 （CTRL-0）

w,b	前进，后退一个单词的开始位置
W,B	前进，后退一个非空区域的开始位置

e,ge	前进，后退到下一个单词的结尾
E,gE	前进，后退到下一个非空区域的结尾

fc,Fc	在当前行向前，向后查找字符c，光标停留在被搜索字符上
tc,Tc	在当前行向前，向后查找字符c，光标停留在字符之前

;	重复上一次的fc,Fc,tc,Tc命令指定次数
,	重复上一次的fc,Fc,tc,Tc相反的命令指定次数

%	跳转到与选中括号对应的括号处，如果当前没有选中括号，则跳转到下一个括号的对应括号处

G	跳转到指定行，13G将跳转到13行，没有参数的G将跳转到最后一行
gg	跳转到文件第一行

``	跳转到上一次行跳转前所在的光标位置
''	同上，但将光标移到了行首

mx	标记当有光标所在的位置为名称x（可任选一个字母为标记名称）
`x	跳转到标记x
'x	跳转到标记x所在行的行首

CTRL-O	在行跳转历史记录中向前跳转

CTRL-I	在行跳转历史记录中向后跳转
TAB	同上



H	移动到屏幕第一行（Home）
L	移动到屏幕最后一行（Last）
M	移动到屏幕中间一行（Middle）

50%	跳转到文件最中间一行，40%将跳转到文件40%的地方（以行为单位）


$	移动到行尾，可计数，6$表示移动到下面第6行的行尾
^	移动到行首第一个非空字符前，不可计数
0	移动到行首，不可计数

CTRL-U	向上滚动半屏
CTRL-D	向下滚动半屏

CTRL-E	向下滚动一行
CTRL-Y	向上滚动一行

CTRL-F	向前滚动一屏
CTRL-B	向后滚动一屏

zz	将当前行置于屏幕中间
zt	将当前行置于屏幕顶端
zb	将当前行置于屏幕底端

n,N	查找下一个/上一个(根据使用的?和/不同，方向也不同)
?,/	向前查找，向后查找

*,#	向前，向后查找当前光标所在的单词
g*,g#	向前，向后查找包含当前光标所在单词字符的内容

d	删除命令，后跟一个移位命令，如dl会将光标左边的字符删掉，d$会从光标当前的位置删到行尾,而不同的是dd将删除整行
c	修改命令，与d类似，只是会进行Insert模式,cw等同于dwi,不同的是cc用来改变整行，但会保留开头的缩进
x	删除光标左边的字符，等同于dl
X	删除光标右边的字符，等同于dh
D	删除于行尾，等同于d$
C	修改于行尾的内容,等同于c$
s	修改一个字符，等同于cl
S	修改一整行，等同于cc
r	替换一个字符，但操作完成后仍然停留在Normal模式（s命令会进入Insert模式）
R	替换模式，新输入的字符会替换原位的字符，只有换行会产生一个新行
.	重复执行上一次的改动(不包含u,CTRL-R命令)

v	进入可视模式，可连续选择文本
V	进入可视模式，用于连续选择整行,左右移位将不可用
CTRL-V	进入可视模式，用于连续选择矩形文本块进行列操作
v_o	可视模式下将光标跳到被选文本的另一端
v_O	可视块选择模式下将光标移到同一行的两角（o是移到对角上）

p	将寄存器中的文本放在光标之后
P	同上，但将放到光标之前

y	复制，后跟移位命令，可以使用p／P输出，yy复制整行
Y	复制整行

aw	一个单词对象，可用在d，y等命令中，也可用在v模式下用于选择一个单词
iw	同上，但包含空格
is	一个句子对象，用法同上
as	一个句子对象，但与is不同的是，as包含句子后面的空格

~	改变选中文本的大小写

ex command
-------------------------------------------------------------------------------------
q	退出
w	写文件（保存文件）
x	保存并退出
e	重新载入文件
cmd!	强制执行cmd
!shell	执行shell命令


option
-------------------------------------------------------------------------------------
(set选项名后面跟＝号表示需要赋一个值。没有的表示是布尔值，只需要在命令前加no即可取消此布尔值)
set opt&		将选项opt还原到默认值
set opt?		显示布尔设置opt的值(opt指某设置的名字)
set number/nu		打开关闭行号
set ruler		在屏幕右下角显示当前当光标的位置
set ignorecase/ic	设置搜索是否忽略大小写
set filetype/ft=		设置文件类型（语法高亮类型）（set ft则自动识别文件类型）


set hlsearch/hls	设置搜索时高亮
nohls/nohlsearch	设置当前一次搜索不高亮

set incsearch		是否键入时即进行搜索
set wrapscan		搜索到文件结尾时再从头开始查找
set compatible		是否以兼容模式运行
set backspace/bs=	设置Insert模式下退格键可以删除的内容 set backspace=indent,eof,start
set autoindent/ai	设置自动缩进
set backup		设置修改文件时自动备份
set history=		设置命令（包含搜索命令和行尾命令）历史记录长度
set wrap		设置是否自动换行
set whichwrap/ww=	设置哪些命令（键）在移到行的开头和末尾时自动移到上一行或下一行
set scrolloff/so=	设置光标在距屏幕上下有多少行时自动发生上下滚动
set sidescroll/ss=	设置光标左右移动移出屏幕时（必须wrap选项被关闭）文本左右移动多少列,设为0会自动滚动一半
set showcmd/sc		设置在Normal模式下输入命令时是否在屏幕下面显示输入的键

map K CMD		将键K里映射为命令CMD（Normal模式），如map MM 50%将使得在Normal模式下按两次M会跳到文件中间，不带参数时将显示所有的映射
edit			编辑文件
options			显示及设置set选项
goto			跳转到文件指定字节处
scriptnames		显示VIM相关设置脚本的路径
jumps			显示跳转历史记录
marks			显示用m命令标记的标记列表,列表中有如下特殊项
				' 进行此次跳转之前的起跳点
				" 上次编辑该文件时光标最后停留的位置
				[ 最后一次修改的起始位置
				] 最后一次修改的结束位置


syntax enable			启用语法高亮
syntax clear			暂时关闭语法高亮
syntax off				彻底停用语法高亮
colorscheme evening		语法高亮风格（evening为风格名称）




map MM 50%
#将N模式下的MM命令映射为跳转到文件中央
set whichwrap=b,s,<,>,[,],h,l
#设置当<Backspace><Space>,<Left>,<Right>,i_<Left>(指Insert模式),i_<Right>,h,l
#命令（键）移到行首或行尾时自动跳转到另一行
